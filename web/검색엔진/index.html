л“Өм–ҙк°Җл©°

м§ҖлӮң лІҲм—җ мұ…м—җ лҢҖн•ң лҘј лӮЁкёҙ м Ғмқҙ мһҲлӢӨ. мұ…м—җм„ң мҶҢк°ңн•ң лӮҙмҡ©мӨ‘ кІҖмғүм—”м§„м—җ кҙҖн•ң лӮҙмҡ©мқҙ мһҲм—ҲлҠ”лҚ°, мқҙм°ём—җ м•Ңм•„л‘җл©ҙ мўӢмқҖ лӮҙмҡ©л“Өмқ„ лӢӨ м •лҰ¬н•ңлӢӨ.

мқҙ кёҖмқҖ

  • robots.txt
  • sitemap
  • webmaster (google|naver)
  • google analytics
    м—җ кҙҖн•ң лӮҙмҡ©мқ„ лӢӨлЈ¬лӢӨ.

мҡ°м„  л“Өм–ҙк°Җкё° м „м—җ м§ҖкёҲ ліҙкі мһҲлҠ” мқҙ лё”лЎңк·ёмқҳ кө¬м„ұм—җ лҢҖн•ҙ м•Ңм•„ліҙлҸ„лЎқ н•ҳмһҗ.

Github Pages мң„м—җм„ң мҢ©мҢ© лҸҢм•„к°Җкі  мһҲлҠ” лё”лЎңк·ё

мқҙ лё”лЎңк·ёлҠ” Github Pages мң„м—җм„ң лҸҷмһ‘н•ҳкі  мһҲмңјл©° Static Site GeneratorлЎң hexoлҘј мӮ¬мҡ©н•ҳкі  мһҲлӢӨ.
(мӢӨм ңлЎң мӣ№нҺҳмқҙм§ҖлҘј мғқм„ұн•ҳкё° мң„н•ҙ мӮ¬мҡ©н•ҳлҠ” мһ‘м—…кіөк°„мқҖ н•ҙлӢ№ л ҲнҸ¬м§ҖнҶ лҰ¬м—җм„ң нҷ•мқён•  мҲҳ мһҲлӢӨ)

н•ҙлӢ№ мһ‘м—…кіөк°„(лЎңм»¬м—җ мһҲмқҢ)м—җм„ң мӣ№нҺҳмқҙм§ҖлҘј л№Ңл“ңн•ҳм—¬ н…ҢмҠӨнҠёлҘј н•ң нӣ„ github pagesлһ‘ м—°лҸҷлҗң л ҲнҸ¬м§ҖнҶ лҰ¬(PngWnA/PngWnA.github.io)м—җ мһҗлҸҷмңјлЎң л°°нҸ¬лҗҳлҠ” нҳ•мӢқмңјлЎң мҡҙмҳҒмқ„ н•ҳкі  мһҲлӢӨ.

л№„мҠ·н•ң нҷҳкІҪмңјлЎң мһҗмӢ мқҳ мӣ№мӮ¬мқҙнҠёлҘј л°°нҸ¬н•ңлӢӨл©ҙ мқҙ кёҖм—җ мһҲлҠ” лӮҙмҡ©л“Өмқ„ л№ лҘҙкІҢ л”°лқјн•  мҲҳ мһҲмқ„ кІғмқҙлқј мғқк°Ғн•ңлӢӨ.
(м•„л§Ҳ л¬ҙлЈҢ нҳёмҠӨнҢ…мқ„ м“ҙлӢӨл©ҙ FTPлҘј мқҙмҡ©н•ҳм—¬ мҳ¬лҰ¬лҠ” кІғ ләҙкі  лӢӨлҘё м җмқҙ нҒ¬кІҢ м—Ҷмқ„ кІғ к°ҷм§Җл§ҢвҖҰ)

robots.txt

(м•„мЈј к°Җм•„лҒ” мҙҲліҙм Ғмқё CTF л¬ём ңм—җ л“ұмһҘн•ҳлҠ”) robots.txtлҠ” мӮ¬лһҢмқҙ мқҪлҠ” нҢҢмқјмқҙ м•„лӢҲлқј кІҖмғүм—”м§„м—җм„ң мғүмқёмқ„ мғқм„ұн•ҳкё° мң„н•ң лҙҮмқҙ мқҪлҠ” нҢҢмқјмқҙлӢӨ.
кІҖмғүм—”м§„м—җ лҙҮл“ӨмқҖ н•ҳмқҙнҚјл§ҒнҒ¬лҘј л”°лқј л– лӢӨлӢҲл©ҙм„ң лӢӨл…Җк°„ мӮ¬мқҙнҠёл“Өмқҳ мғүмқё(index)лҘј мғқм„ұн•ҳлҠ”лҚ°, лҸ„л©”мқёмқҳ мөңмғҒмң„ нҸҙлҚ”(/)м—җ robots.txtк°Җ мЎҙмһ¬н•ҳл©ҙ н•ҙлӢ№ нҢҢмқјмқ„ м°ёкі н•ҙ м ‘к·јн•ҙлҸ„ лҗҳлҠ” нҢҢмқјл“Өмқ„ нҢҢм•…н•ңлӢӨ.

мқҙ лё”лЎңк·ёмқҳ robots.txtлҠ” лӢӨмқҢкіј к°ҷмқҙ кө¬м„ұлҗҳм–ҙ мһҲлӢӨ.

н•ҙлӢ№ robots.txtлҠ” лӘЁл“ (*) User-agent(мЈјлЎң лҙҮ)к°Җ мқҙ лҸ„л©”мқё м•„лһҳ лӘЁл“  нҢҢмқј(/)мқ„ нғҗмғүн•ҙлҸ„ лҗңлӢӨкі  м„ м–ён•ңлӢӨ.
(лҳҗн•ң, Sitemapмқҳ мң„м№ҳ м—ӯмӢң н‘ңмӢңн•ҳлҠ”лҚ°, sitemapмқҖ нӣ„м—җ мӮҙнҺҙліҙмһҗ)

л§Ңм•Ҫ ліёмқёмқҙ л„ӨмқҙлІ„к°Җ мӢ«м–ҙм„ң л„ӨмқҙлІ„мқҳ кІҖмғүм—”м§„ лҙҮ(Yeti)мқҳ кІҖмғүмқ„ л§үкі мһҗ н•ңлӢӨ? к·ёл ҮлӢӨл©ҙ лӢӨмқҢкіј к°ҷмқҖ лӮҙмҡ©мқ„ robots.txtмқҳ м•һм—җ м„ м–ён•ҳл©ҙ лҗңлӢӨ.

мқҙ лё”лЎңк·ём—җ мң„мҷҖ к°ҷмқҖ лӮҙмҡ©мқҙ мһҲм—ҲлӢӨл©ҙ л„ӨмқҙлІ„мқҳ YetiлҠ” мқҙ лё”лЎңк·ёлҘј нғҗмғүн•ҳм§Җ м•ҠлҠ”лӢӨ.

лҳҗлҠ” нҠ№м • л””л үнҶ лҰ¬ м•„лһҳ мһҲлҠ” мӣ№нҺҳмқҙм§Җл“Өмқҳ кІҖмғүл§Ң л§үкі мһҗ н•ҳл©ҙ лӢӨмқҢкіј к°ҷмқҙ м„ м–ён•ҳл©ҙ лҗңлӢӨ.

мқҙл ҮкІҢ м„ м–ён•ң кІҪмҡ°м—җлҠ” кө¬кёҖлҙҮмқҖ admin нҸҙлҚ” м•„лһҳ мһҲлҠ” лӮҙмҡ©мқ„ нғҗмғүн•ҳм§Җ м•ҠлҠ”лӢӨ.

н•ҙлӢ№ лӮҙмҡ©л“Өмқ„ лӢӨ н•©м№ҳл©ҙ мқҙлҹ¬н•ң к·ңм№ҷмқҙ нғ„мғқн•ҳкІҢ лҗңлӢӨ.

мқҙлҹ° л°©мӢқмңјлЎң robots.txtлҘј мһҳ мҲҳм •н•ҙм„ң мөңмғҒмң„ нҸҙлҚ”(/) м•„лһҳ л„Јмңјл©ҙ лҗңлӢӨ.

Github pagesлҘј мӮ¬мҡ©н•ҳлҠ” кІҪмҡ°м—җлҠ” л ҲнҸ¬м§ҖнҶ лҰ¬ мөңмғҒмң„ нҸҙлҚ”м—җ robots.txtлқјлҠ” мқҙлҰ„мңјлЎң нҢҢмқјмқ„ л„Јкі  add -> commit -> push н•ҙмЈјл©ҙ лҗҳкі 
л¬ҙлЈҢ нҳёмҠӨнҢ…мқ„ мӮ¬мҡ©н•ҳл©ҙ FTP л“ұмқ„ мӮ¬мҡ©н•ҙм„ң мҳ¬лҰ¬л©ҙ лҗңлӢӨ.

sitemap

sitemapмқҖ н•ҙлӢ№ мӣ№мӮ¬мқҙнҠёмқҳ кө¬мЎ°лҘј м„ӨлӘ…н•ҳлҠ” нҢҢмқјмқҙлқјкі  м„ӨлӘ…н•ҳл©ҙ мүҪкІҢ мқҙн•ҙн•  мҲҳ мһҲмқ„ кІғмқҙлӢӨ.
(л§җ к·ёлҢҖлЎң siteмқҳ mapмқҙлқј лҚ” м„ӨлӘ…н•  л°©лІ•мқҙ м—ҶлӢӨвҖҰ)
ліҙнҶө XMLлЎң мЈјлҠ”лҚ°, мқҙ лё”лЎңк·ёмқҳ sitemapлҘј мӮҙнҺҙліҙмһҗ.

sitemapмқҳ indexлҘј лҸ„л©”мқё мөңмғҒмң„м—җ мҳ¬л ӨлҶЁлҠ”лҚ°, мң„мҷҖ к°ҷмқҙ м„ңлІ„м—җ requestлҘј лӮ лҰ¬л©ҙ нҷ•мқён•  мҲҳ мһҲлӢӨ.

лӢӨмқҢмқҖ responseлЎң л“Өм–ҙмҳӨлҠ” sitemap.xmlмқҙлӢӨ.

sitemapindex м•„лһҳ м—¬лҹ¬к°ңмқҳ sitemapмқҙ мЎҙмһ¬н•ҳлҠ”лҚ° мқҙ мӨ‘м—җ н•ҳлӮҳлҘј м°Қм–ҙм„ң л“Өм–ҙк°Җліҙмһҗ.

responseлЎң мҳЁ xmlмқҙ мӢӨм ң sitemapмқҳ лӮҙмҡ©мқёлҚ°, лҢҖл¶Җл¶„ urlset м•„лһҳ urlл“Өмқҙ мһҲкі , н•ҙлӢ№ urlм—җ лҢҖн•ң м„ӨлӘ…(loc:мң„м№ҳ, lastmod:л§Ҳм§Җл§ү мҲҳм •мӢңк°„)мқҙ мЎҙмһ¬н•ңлӢӨ.
лҢҖл¶Җл¶„мқҳ sitemapмқҖ м Җлҹ° нҳ•мӢқмңјлЎң кө¬м„ұмқҙ лҗңлӢӨ.

мқҙм •лҸ„л§Ң м•Ңм•„лҸ„ мҲҳлҸҷмңјлЎң sitemapмқ„ мһ‘м„ұн•  мҲҳ мһҲлӢӨ. м •нҷ•н•ң лӘ…м„ёлҠ” sitemap.orgм—җм„ң нҷ•мқён•  мҲҳ мһҲлӢӨ.

мҲҳлҸҷмңјлЎң л§Ңл“Өкі  мӢ¶мңјл©ҙ мң„мҷҖ к°ҷмқҖ xmlнҳ•мӢқмқ„ м§Җмјңм„ң л§Ңл“Өл©ҙ лҗҳм§Җл§Ң ліҙнҶө мһҗлҸҷмңјлЎң мғқм„ұн•ҙмЈјлҠ” л°©лІ•мқҙ мЎҙмһ¬н•ҳкё° л§Ҳл ЁмқҙлӢӨ.
hexoлҠ” Sitemap GeneratorлҘј м„Өм№ҳн•ҳл©ҙ мӣ№мӮ¬мқҙнҠёк°Җ мғқм„ұлҗ  л•Ң мһҗлҸҷмңјлЎң sitemap.xmlмқ„ мғқм„ұн•ҙмӨҖлӢӨ.

hexoмқҳ Sitemap GeneratorлҘј м„Өм№ҳн•ҳкі мһҗ н•ҳл©ҙ

лЎң м„Өм№ҳн•ҳкі  hexo нҸҙлҚ”мқҳ мөңмғҒмң„м—җ мһҲлҠ” _config.ymlм—җ

лҘј м¶”к°Җн•ҙмЈјл©ҙ м•Ңм•„м„ң мғқм„ұн•ҙмӨҖлӢӨ.

hexoлҘј м“°м§Җ м•ҠлҠ”лӢӨл©ҙ мһҗлҸҷмңјлЎң sitemapмқ„ мғқм„ұн•ҙмЈјлҠ” м„ңл№„мҠӨлҘј мӮ¬мҡ©н•ҳкұ°лӮҳ мҲҳлҸҷмңјлЎң мғқм„ұн•ҙм„ң мҳ¬лҰ¬мһҗ.

webmaster

robots.txtмҷҖ sitemap.xmlмқҙ мҷ„м„ұлҗҳм—ҲлӢӨ. мқҙм ң кІҖмғүм—”м§„мқҙ мҡ°лҰ¬ мӮ¬мқҙнҠёлҘј ліҙлӢӨ нҺён•ҳкІҢ нғҗмғүн•  мҲҳ мһҲлӢӨ!

н•ҳм§Җл§Ң мҲҳлҸҷм ҒмңјлЎң кІҖмғүм—”м§„мқҙ мҡ°лҰ¬ мӮ¬мқҙнҠёлҘј м°ҫм•„м„ң м—ҙмӢ¬нһҲ нғҗмғүн•ҙмЈјкё°лҘј л°”лқјлҠ” кІғліҙлӢӨ лҚ” л№ лҘё кІғмқҖ мҡ°лҰ¬к°Җ лҠҘлҸҷм ҒмңјлЎң кІҖмғүм—”м§„м—җ мҡ°лҰ¬ мӮ¬мқҙнҠёлҘј л“ұлЎқн•ҳлҠ” л°©лІ•мқҙлӢӨ.

л‘җ к°Җм§Җ л°©лІ•мқ„ лӢӨ мӮ¬мҡ©н•ҳл©ҙ мҷ м§Җ кІҖмғүм—”м§„мқҙ мҡ°лҰ¬ лё”лЎңк·ёлҘј лҚ” мўӢм•„н•ҙмӨ„ кІғ к°ҷлӢӨ. л‘ҳ лӢӨ н•ҙліҙмһҗ.

google

Google Search Consoleм—җ л“Өм–ҙк°Җмһҗ.

лӢӨмқҢкіј к°ҷмқҖ нҷ”л©ҙмқҙ мҡ°лҰ¬лҘј л°ҳкІЁмӨҖлӢӨ.

Google Search Console

мӣ№мӮ¬мқҙнҠёлҘј кө¬кёҖ м„ңм№ҳм—җ л“ұлЎқн•ҳкё° мң„н•ҙм„ңлҠ” мӮ¬мқҙнҠёмқҳ мҶҢмң к¶Ңмқ„ мҰқлӘ…н•ҙм•ј н•ҳлҠ”лҚ°, мЈјлЎң лӢӨмқҢкіј к°ҷмқҖ л°©мӢқмңјлЎң мқҙлЈЁм–ҙм§„лӢӨ.

  • DNS Recordм—җ нҠ№м • лӮҙмҡ© м¶”к°Җ (мҷјмӘҪ)
  • мӣ№нҺҳмқҙм§Җм—җ нҠ№м • HTML нҢҢмқј м—…лЎңл“ң (мҳӨлҘёмӘҪ)
  • мӣ№нҺҳмқҙм§Җм—җ нҠ№м • meta нғңк·ё м¶”к°Җ (мҳӨлҘёмӘҪ)
  • м• л„җлҰ¬нӢұмҠӨ м—°лҸҷ (мҳӨлҘёмӘҪ)

к°Җл№„м•„лӮҳ лӢӨлҘё лҸ„л©”мқё м„ңл№„мҠӨлҘј мқҙмҡ©н•ҙм„ң лҸ„л©”мқёмқ„ кө¬л§Өн–ҲлӢӨл©ҙ мҷјмӘҪ л°©лІ•мқ„ мӮ¬мҡ©н•ҙлҸ„ лҗңлӢӨ.
н•ҳм§Җл§Ң мқҙ лё”лЎңк·ёлҠ” Github Pages мң„м—җ мҳ¬лқјк°Җ мһҲмңјлӢҲ мҳӨлҘёмӘҪ л°©лІ•мӨ‘ HTML нҢҢмқјмқ„ м—…лЎңл“ң н•ҳлҠ” л°©мӢқмқ„ мӮ¬мҡ©н•ҳлҸ„лЎқ н•ҳмһҗ.
(л¬ҙлЈҢ нҳёмҠӨнҢ…мқҳ кІҪмҡ°м—җлҸ„ DNS л ҲмҪ”л“ңлҘј ліҖкІҪн•  мҲҳ м—ҶлҠ” кІҪмҡ°к°Җ л§ҺмңјлӢҲ н•ҙлӢ№ л°©лІ•мқ„ л”°лқјн•ҳл©ҙ лҗңлӢӨ)

мҡ°м„  мҳӨлҘёмӘҪм—җ ліёмқёмқҳ мӣ№мӮ¬мқҙнҠёмқҳ URLмқ„ мһ…л Ҙн•ҳл©ҙ мҶҢмң к¶Ң нҷ•мқё л°©лІ•мқҙ лӮҳмҳӨлҠ”лҚ°, мІ«лІҲм§ё л°©лІ•мқ„ мӮ¬мҡ©н•ҳкё°лЎң н–ҲмңјлҜҖлЎң мІ«лІҲм§ё л°©лІ•м—җ лҢҖн•ҙм„ң м•Ңм•„ліҙмһҗ.

HTML нҢҢмқј м—…лЎңл“ң к¶Ңн•ңмқ„ нҶөн•ң мҶҢмң к¶Ң мқёмҰқ

нҠ№м • нҢҢмқјмқ„ мөңмғҒмң„ нҸҙлҚ”м—җ мҳ¬лҰҙ кІғмқ„ мҡ”кө¬н•ҳкі  мһҲлҠ”лҚ°, н•ҙлӢ№ html нҢҢмқјмқ„ лӢӨмҡҙл°ӣм•„м„ң мҳ¬лҰ¬л©ҙ лҒқлӮңлӢӨ.

н•ҙлӢ№ нҢҢмқјм—җ л¬ҙмҠЁ лӮҙмҡ©мқҙ мһҲлҠ”м§Җ к¶ҒкёҲн•  кІғ к°ҷмқҖлҚ°, лі„кұ° м—ҶлӢӨ.

мқёмҰқлӮҙмҡ© н•ҳлӮҳл§Ң л“Өм–ҙмһҲлҠ” нҢҢмқјмқҙлӢӨ.
(м°ёкі лЎң л„ӨмқҙлІ„лҸ„ кұ°мқҳ лҳ‘к°ҷмқҖ л°©мӢқмқ„ мҶҢмң к¶Ң нҷ•мқём—җ мӮ¬мҡ©н•ңлӢӨ)

н•ҙлӢ№ нҢҢмқјмқ„ м—…лЎңл“ңн•ҳкі  нҷ•мқёмқ„ лҲ„лҘҙл©ҙ мҶҢмң к¶Ң нҷ•мқёмқҙ лҒқлӮҳкі  м„ңм№ҳ мҪҳмҶ”мқҙ м—ҙлҰ°лӢӨ.
